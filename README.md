# 뉴로시민 | Everyday Neuroscience Lab

뇌과학을 시민의 일상 언어로 설명하는 공익 뇌과학 리터러시 GitHub Pages 아카이브입니다.

- **다루는 질문:** 배움, 집중, 기억, 수면, 감정, 습관, 관계, 디지털 환경
- **편집 원칙:** 연구 근거 우선, 쉬운 설명, 과장 없는 해석, 의료적 진단·치료 단정 금지
- **공개 원칙:** 후보군·점수·선정 과정·자동화 정보는 내부 편집용이며 공개 글에 노출하지 않음
- **배포:** GitHub Pages

## 자동 발행 흐름

1. 매일 최신 연구·공식 자료를 확인
2. 새로움·흥미성·교육/학습 적합성·공익성·검증성 기준으로 내부 검토
3. 검증 가능한 주제 1개를 선택해 `_posts/YYYY-MM-DD-slug.md` 작성
4. GitHub Pages 빌드·라이브 URL·이미지·구조화 데이터 검증
