# Everyday Neuroscience Lab

부모·교사·학습자를 위한 공익형 뇌과학 지식 GitHub Pages 블로그입니다.

- **운영 원칙:** 매일 뇌과학·학습과학·멘탈케어 후보 주제 5개를 조사하고, 가장 신선하고 흥미롭고 공익적인 1개를 선정해 발행
- **글쓰기 기준:** AEO·GEO·EEAT, 일반인 눈높이, 과장 없는 연구 해설
- **배포:** GitHub Pages

## 자동 발행 흐름

1. 매일 07:30 KST 뇌과학 후보 주제 5개 리서치
2. 새로움·흥미성·교육/학습 적합성·공익성·검증성 점수화
3. 최고 점수 주제 1개 선정
4. `_posts/YYYY-MM-DD-slug.md` 작성
5. GitHub Pages 발행
